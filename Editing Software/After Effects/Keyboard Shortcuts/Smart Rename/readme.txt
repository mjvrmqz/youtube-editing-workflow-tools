Smart Rename
Syncs naming between layers and their source items in the project panel. If layers are selected in a composition, their source items will be renamed to match the layer name. If items are selected in the project panel, their corresponding timeline layers will be renamed to match the source name.

Constraints
Requires either layers selected in an active composition or items selected in the project panel
AE CC 2014 or later

Notes
On first run you will be prompted to confirm. Check "Never show again" to skip this in the future.
A "Reset" file is included to restore default settings.
