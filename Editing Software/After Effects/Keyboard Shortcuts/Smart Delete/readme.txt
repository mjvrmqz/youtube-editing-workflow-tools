Smart Rename
Syncs naming between layers and their source items in the project panel. If layers are selected in a composition, their source items will be renamed to match the layer name. If items are selected in the project panel, they will be renamed to match their corresponding layer name in any composition.

Constraints
Requires either layers selected in an active composition or items selected in the project panel
AE CC 2014 or later

Notes
When renaming from the project panel, the source will be renamed to match the first layer found using it across all compositions.
